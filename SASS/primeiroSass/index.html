para salvar o sass como dependencia de desenvolvimento

npm i sass --save-dev

para compilar o codigo .sass

sass --watch sass/style.sass:style/style.css

coloque o script abaixo antes

para .sass

"scripts": { "sass": "sass --watch ./sass/style.sass ./style/style.css" }

para .scss

"scripts": { "sass": "sass --watch ./sass/style.scss ./style/style.css"}
